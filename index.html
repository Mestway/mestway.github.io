About

I’m a second-year computer science PhD student in the PLSE group at the University of Washington, advised by Ras Bodik and Alvin Cheung. I'm working on program synthesis algorithms and their applications on end-user programming.

Prior to UW, I studied in Peking University. I was lucky to work with Zhenjiang Hu and Yingfei Xiong on designing transformation languages for Java API adaptation on users' side.

Projects

Recently I'm involved in the following two projects:

  • Scythe: Synthesizing SQL queries from input-output examples.
  • Cosette: Formal reasoning of SQL qurey equivalence.
