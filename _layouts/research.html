Research

My main interest lies in programming languages research, especially type theory, language design and program synthesis. Besides, I'm also interested in software engineering and visualization.

Publications

  • SWIN: Towards Type-Safe Java Program Adaptation between APIs
    J.Li, C.Wang, Y.Xiong, Z.Hu.
    To appear, ACM SIGPLAN 2015 Workshop on Partial Evaluation and Program Manipulation (PEPM'15)
    Mumbai, India, Jan 12-13, 2015
    We design a domain specific language SWIN to help software developer adapt their code when API upgrades are needed. SWIN can miantain the well-typedness of the program transformation process: the transformed program will remain well-typed if the SWIN program is well formed.
  • Composable and Hygienic Typed Syntax Macros
    C.Omar, C.Wang, J.Aldrich.
    To appear, 30th ACM Symposium on Applied Computing (SAC 2015)
    Salamanca, Spain, April 13-17, 2015
    We proposed an innovative way to support language extension through typed syntax macros (TSMs) based on TSL mechenism. TSMs provide a safe way to extend the host language by allowing users to add parsing rules in the definition of macros.

In Progress

  • Safely context-sensitive transformation language for Java API adaptation
    with J.Li, Y.Xiong, Z.Hu.
    (In progress)
    We are trying to enhance our SWIN language to support context sensitive transformation for API adaptation while remain safety: types and dependencies will not get into trouble.
