Research

My main interests is in programming language, especially type theory, domain-specific language and program manipulation. Besides, I'm also quite interested in software engineering and visualization.

  • Safely context-sensitive transformation language for Java API adaptation
    with J.Li, Y.Xiong, Z.Hu.
    (In progress)
    We are trying to enhance our SWIN language to support context sensitive transformation for API adaptation while remain safety: types and dependencies will not get into trouble.
  • SWIN: Type-safe transforamtion language for Java API adaptation
    J.Li, C.Wang, Y.Xiong, Z.Hu.
    To appear, ACM SIGPLAN 2015 Workshop on Partial Evaluation and Program Manipulation (PEPM'15)
    Mumbai, India, Jan 12-13,2015
    We design a domain specific language SWIN to help software developer adapt their code when API upgrade is needed. SWIN can miantain the well-typedness of the program transformation process: the transformed program will remain well-typed when the SWIN program is well formed.
  • Composable and hygienic syntax typed macros
    C.Omar, C.Wang, J.Aldrich.
    (Submitted)
    We proposed an innovative way to support language extension through typed syntax macros (TSMs) based on TSL mechenism. TSMs provide a safe way to extend the host language by allowing users to add parsing rules in the definition of macros.
  • IEEE VAST Challenge 2014, mini challenge 1
    C.Wang, S.Chen, Z.Miao, X.Yuan.
    IEEE VAST Challenge 2014 - Award for Excellent Detailed Analysis, Paris, France
    Provide Story Explorer system to solve the challenge of news, email and entity relationship analysis from IEEE VAST Challenge 2014, Mini Challenge 1.
